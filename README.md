## Overview

## Release Notes

#### 0.1.0

## Getting Started

**Via Gradle**

Add the jitpack repo to your root build.gradle:

```
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```

Add the dependency:

```
dependencies {
  implementation 'com.github.Simbule:KuponCentro-Android-SDK:0.1.0'
}
```
